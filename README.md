Challenge 1 - Automatizacion de IoCs - Campeonato Nacional de Ciberseguridad 

## Dependencias
Python >= 3.8

## Instalacion
```shell
pip install -r requeriments.txt
```

## Uso
```shell
python main.py -i <inputfile> -o <outputfie>
```
